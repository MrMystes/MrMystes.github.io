追书神器api分析

搜索书籍 5702151c1c8e4db9483762d6

GET http://api.zhuishushenqi.com/book/fuzzy-search?query={bookname}&start=0&limit=100

返回值

​

书籍主页信息

GET http://api.zhuishushenqi.com/book/{bookId}

返回值

​

根据bookId获取源信息

GET http://api.zhuishushenqi.com/toc?view=summary&book={bookId}

返回值

​

获得书籍目录

GET http://api.zhuishushenqi.com/toc/{sourceId}?view=chapters // sourceId是对应的书源的id

返回值

​

获得章节内容

GET http://chapter2.zhuishushenqi.com/chapter/{link} // link是网页链接

返回值
